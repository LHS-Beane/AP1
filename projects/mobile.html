AP Mobile Equilibrium Project

Design a hanging mobile and prove—using AP Physics 1 reasoning—that every tier satisfies translational equilibrium and rotational equilibrium.

🎯 The Challenge

You will design and construct a multi-tier hanging mobile that is stable at rest. The mobile must be modeled as a set of rigid bodies and analyzed using both equilibrium conditions:

ΣF = 0   and   Στ = 0
AP Skill: Modeling AP Skill: Torque AP Skill: Error Analysis Build: Craftsmanship

AP Level Expectations

  • Symbolic first: derive equations before plugging in numbers.
  • FBDs required: one per tier (rigid body, not just point masses).
  • Axis choice: must be stated and justified.
  • Validation: compare prediction vs measurement and compute percent error.

✅ What You’ll Learn

  • How to treat an extended object as a rigid body
  • How the center of mass relates to stability
  • Why ΣF = 0 alone does NOT prevent rotation
  • How torque depends on distance and angle: τ = rF\sin\theta

🛠️ Build Specifications

Design Criteria

  • Minimum: 3 freely-swinging horizontal arms (3+ tiers)
  • Size limit: ≤ 50 cm wide, 50 cm deep, 100 cm tall
  • Safety: no fragile items, hooks, or anything hazardous/inappropriate
  • Original theme: artistic OR physics-based

AP Upgrade Requirements

  • At least one arm must have non-negligible rod mass (you must include it in calculations).
  • At least one tier must be asymmetric (not “equal masses equal distances”).
  • Arms must hang approximately horizontal at rest (you’ll address small deviations in error).
Pro Tip: Start with small masses and designs that allow easy adjustment. Balance comes from changing either mass or lever arm distance.

📦 Suggested Materials

  • Dowel rods / wooden skewers / lightweight rulers
  • String, thread, fishing line (thin)
  • Paper clips, binder clips, small hooks (safe)
  • Washers / nuts / bolts / coins for adjustable masses
  • Digital scale (grams), meter stick, ruler, graph paper

📘 AP Physics Requirements

You must provide the following for each tier:

1) System Definition + FBD

  • Define the system: “Tier 2 arm + attached masses”
  • External forces: tensions, weight(s)
  • Include rod weight if required on that tier
  • Label all distances from pivot

2) Translational Equilibrium

ΣFy = 0

Use this to relate the support tension to total weight.

3) Rotational Equilibrium

Στ = 0

Choose a pivot (usually the suspension point). State sign convention and justify why that pivot simplifies. If forces are vertical and arms are horizontal, you can often use |&tau| = rF.

4) Validation + Error Analysis

  • Predict pivot location(s) before building
  • Measure actual distances after building
  • Compute percent error and discuss sources

AP-Style Concept Questions (Required)

  1. If one hanging mass is doubled, what change must be made to restore balance? Explain without full recalculation.
  2. If the mobile were taken to the Moon, would it still balance? Justify using equilibrium equations.
  3. If an arm tilts 10° from horizontal, which part of your torque model changes? Explain conceptually.

🧮 Calculations Toolkit

Torque Reminder

τ = rF\sin\theta
  • r = distance from pivot to the line of action (m)
  • F = force magnitude (N)
  • θ = angle between r and F

Typical Mobile Simplification

If the arm is horizontal and forces are vertical, then θ = 90° and:

τ = rF

Two-Mass Balance (One Tier)

For two hanging masses on opposite sides of a pivot:

mLg rL = mRg rR  →  mL rL = mR rR

Include Rod Mass (AP Upgrade)

If the rod has mass mrod and its center of mass is at distance rcm from the pivot:

Στ = (mLg rL) - (mRg rR) - (mrodg rcm) = 0
