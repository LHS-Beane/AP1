Unit 1: Kinematics

Mastering the description of motion: position, velocity, and acceleration in 1D and 2D space.

Key Visual Concepts

1. Position vs. Time ($x$ vs $t$)

The slope of this graph represents velocity ($v$).

  • Straight line = Constant Velocity
  • Curved line = Acceleration
2. Projectile Motion

Motion in 2 dimensions is independent.

  • x-axis: Constant velocity ($a_x = 0$)
  • y-axis: Accelerated by gravity ($a_y = -g$)
(Vector Components Diagram)
3. Vectors & Scalars

Displacement ($\Delta x$) is a vector; Distance ($d$) is a scalar.

Split vectors into components:

$v_x = v \cos(\theta)$ | $v_y = v \sin(\theta)$

The "Big 3" Kinematic Equations

Note: These are only valid when acceleration ($a$) is constant.

Velocity as a function of time

$$v_x = v_{x0} + a_x t$$

Use when missing $\Delta x$

Position as a function of time

$$x = x_0 + v_{x0} t + \frac{1}{2}a_x t^2$$

Use when missing $v_f$

Velocity independent of time

$$v_x^2 = v_{x0}^2 + 2a_x(x - x_0)$$

Use when missing time ($t$)

Unit Labs & Investigations

1. The Buggy Lab (Constant Velocity)

Objective: Establish the relationship between position and time for an object with zero acceleration.

Data Analysis:

  • Plot $x$ (y-axis) vs $t$ (x-axis).
  • Equation of the line: $x = mt + b$.
  • Slope ($m$) corresponds to velocity ($v$).
  • Y-intercept ($b$) corresponds to initial position ($x_0$).
2. Cart on a Ramp (Uniform Acceleration)

Objective: Determine the acceleration due to gravity on an incline.

Linearization:

Since $x \propto t^2$, plotting $x$ vs $t$ yields a curve. To linearize:

  • Plot $x$ on y-axis and $t^2$ on x-axis.
  • Slope $= \frac{1}{2}a$. Therefore, $a = 2 \times \text{slope}$.
