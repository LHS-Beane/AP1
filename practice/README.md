# AP Physics 1 Practice Exam Builder

Static GitHub Pages practice site.

Updates in this version:
- Questions without answer keys were removed.
- Practice now displays one question at a time with Previous/Next navigation.
- Students submit once to check answers and receive a score.
- Diagrams/images are embedded directly in `questions.js`, avoiding GitHub's browser upload limit for many individual images.
- Client-side formatting improves common physics notation, including scientific notation, superscripts, unit powers, and common variable subscripts.
